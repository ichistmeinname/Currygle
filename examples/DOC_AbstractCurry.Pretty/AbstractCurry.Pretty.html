Pretty printing of AbstractCurry.

This library provides a pretty-printer for AbstractCurry modules.

Author: Yannik Potdevin

Version: September 2015

options :: Int -> Int -> Qualification -> String -> Options   

Example call: (options pw iw q m rels)

Parameters:

  • pw : the page width to use
  • iw : the indentation width to use
  • q : the qualification method to use
  • m : the name of the current module
  • rels : A list of all to the current module related modules, i.e. the current module itself and all its imports (including prelude). The current module must be the head of that list.
Returns:
options with the desired behavior
